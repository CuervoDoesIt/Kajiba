    **GitHubOps class** (per D-02, all GitHub operations via gh CLI):
    - `__init__(self, upstream_repo: str, timeout: int = 120)` — stores repo and timeout
    - `_run_gh(self, args: list[str]) -> GhResult` — wraps `subprocess.run(["gh"] + args, capture_output=True, text=True, timeout=self._timeout)`. Catches FileNotFoundError (returns GhResult with stderr="gh CLI not found. Install from https://cli.github.com/", returncode=-1). Catches TimeoutExpired (returns GhResult with stderr="Command timed out", returncode=-2).
    - `_run_git(self, args: list[str], cwd: Optional[str] = None) -> GhResult` — same pattern but for `["git"] + args` with optional cwd parameter
    - `check_auth(self) -> GhResult` — `_run_gh(["auth", "status"])`
    - `fork_repo(self) -> GhResult` — `_run_gh(["repo", "fork", self._upstream, "--clone=false"])` (idempotent per Pitfall 1)
    - `clone_fork(self, dest: str, fork_url: str) -> GhResult` — `_run_git(["clone", fork_url, dest])`
    - `pull_latest(self, repo_dir: str) -> GhResult` — `_run_git(["fetch", "origin"], cwd=repo_dir)` then `_run_git(["reset", "--hard", "origin/main"], cwd=repo_dir)`
    - `create_branch(self, repo_dir: str, branch_name: str) -> GhResult` — `_run_git(["checkout", "-b", branch_name], cwd=repo_dir)`
    - `commit_all(self, repo_dir: str, message: str) -> GhResult` — `_run_git(["add", "."], cwd=repo_dir)` then `_run_git(["commit", "-m", message], cwd=repo_dir)`
    - `push_branch(self, repo_dir: str, branch_name: str) -> GhResult` — `_run_git(["push", "-u", "origin", branch_name], cwd=repo_dir)`
    - `create_pr(self, title: str, body: str, head: str, base: str = "main") -> GhResult` — `_run_gh(["pr", "create", "--repo", self._upstream, "--title", title, "--body", body, "--head", head, "--base", base])`
    - `get_username(self) -> GhResult` — `_run_gh(["api", "user", "-q", ".login"])` for determining fork owner

    **Pure functions section:**

    `normalize_model_name(name: str) -> str`:
    Per D-05. Lowercase, replace non-alphanumeric (except hyphens) with hyphens, collapse multiple hyphens, strip leading/trailing hyphens. Use `re.sub`. Dots become hyphens (so "3.5" becomes "3-5").

    `compute_shard_key(record_id: str, num_shards: int = DEFAULT_NUM_SHARDS) -> str`:
    SHA-256 of record_id, first 2 hex chars mod num_shards, return `f"shard_{shard_num:02x}.jsonl"`.

    `compute_record_path(model_name: Optional[str], quality_tier: Optional[str], record_id: str) -> str`:
    Default model_name to "unknown" if None/empty. Default quality_tier to "review_needed" if None/empty. Normalize model_name. Return `f"data/{model_dir}/{tier}/{shard_file}"` using FORWARD SLASHES (per Pitfall 6 — cross-platform). Use string concatenation, NOT pathlib (which would use backslashes on Windows).

    `write_records_to_shards(repo_root: Path, records: list[dict]) -> int`:
    For each record dict: extract model_name from `record.get("model", {}).get("model_name")`, quality_tier from `record.get("quality", {}).get("quality_tier")`, record_id from `record.get("record_id")`. Skip records without record_id (log warning). Compute relative path via compute_record_path. Construct absolute path as `repo_root / rel_path` (using Path for filesystem ops). Create parent directories. Read existing shard file to extract existing record_ids (scan each JSON line for `"record_id"`). Skip duplicate record_ids. Append new record as JSON line. Return count of written records.

    `generate_catalog(repo_root: Path) -> dict`:
    Scan `repo_root / "data"` directory tree. For each `model/tier/shard_*.jsonl` structure: count records by reading lines, compute file sizes, extract average quality score from `quality.composite_score` field, track hardware distribution from `hardware.gpu_name`. Count deletions from `repo_root / "deletions.jsonl"`. Return dict matching the catalog.json schema from research:
    ```python
    {
        "schema_version": SCHEMA_VERSION,
        "generated_at": datetime.now(UTC).isoformat(),
        "total_records": total,
        "total_size_bytes": total_bytes,
        "models": { model_slug: { "display_name": first_seen_name, "tiers": { tier: { "record_count": N, "avg_quality_score": avg, "shards": [filenames], "total_size_bytes": bytes, "last_updated": iso_ts } }, "total_records": N, "hardware_distribution": { gpu_name: count } } },
        "quality_distribution": { "gold": N, "silver": N, "bronze": N, "review_needed": N },
        "deletions_count": N
    }
    ```

    `generate_readme(catalog: dict) -> str`:
    Per D-08. Static template with dynamic stats. Use f-string interpolation (no Jinja2). Static sections: title, license (Apache 2.0), About, Privacy & Scrubbing, How to Use, Deletions, Contributing. Dynamic sections: Dataset Statistics table (Total Records, Models, Last Updated), Quality Distribution table (Tier, Count, Percentage), Model Coverage table (Model, Records, Avg Score, Top Tier). Include `<!-- AUTO-GENERATED -->` marker before dynamic sections.
