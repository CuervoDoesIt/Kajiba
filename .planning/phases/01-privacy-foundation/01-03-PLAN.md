    **Step 1: Update collector.py — Wire privacy into export_record()**

    Add import at top of `src/kajiba/collector.py`:
    ```python

    ```

    Modify `export_record()` method. The current pipeline is:
    1. `_build_record()` -> raw record
    2. `scrub_record()` -> scrubbed record + scrub_log
    3. Attach scrub_log
    4. Compute IDs

    Change to this pipeline order (per Pitfall 5 from RESEARCH.md):
    1. `_build_record()` -> raw record
    2. `scrub_record(record)` -> scrubbed record + scrub_log (PII scrub first)
    3. `anonymize_hardware(scrubbed)` -> hw-anonymized record (per D-08: after PII scrub)
    4. `jitter_timestamp(anonymized)` -> timestamp-jittered record (per D-07)
    5. Read consent level from `record.submission.consent_level` (per D-03) — use the ORIGINAL record's submission, not the scrubbed one, in case scrubbing changed it. Default to `"full"` if submission is None.
    6. `apply_consent_level(jittered, consent_level)` -> consent-stripped record (per D-01: last step)
    7. Attach scrub_log to the final record's submission
    8. Compute IDs

    The updated `export_record()` method body:
    ```python
    def export_record(self) -> KajibaRecord:
        record = self._build_record()

        # Step 1: PII scrub (must be first — scrubs all fields before any are stripped)
        scrubbed, scrub_log = scrub_record(record)

        # Step 2: Hardware anonymization (after scrub, before consent strip)
        anonymized = anonymize_hardware(scrubbed)

        # Step 3: Timestamp jitter
        jittered = jitter_timestamp(anonymized)

        # Step 4: Consent enforcement (last — strips already-clean data)
        consent_level = "full"
        if record.submission and record.submission.consent_level:
            consent_level = record.submission.consent_level
        final = apply_consent_level(jittered, consent_level)

        # Attach scrub log
        if final.submission is None:
            final.submission = SubmissionMetadata()
        final.submission.scrub_log = scrub_log

        # Compute IDs
        final.compute_record_id()
        final.compute_submission_hash()

        return final
    ```
